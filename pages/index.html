Quick tips about programming

Before starting your learning about programming make sure all of this is checked:

1. First analyize the problem and try to think of a solution then solve it!
2. Keep the code as clean as possible to make it readable for anyone.
3. Save as memory space as possible, try the best methods to have less code.
4. Find the programming language or the IDE that fits you the best in work.
5. Make as much reusable code fragments as you can. It makes the work easier.

Hasznos tippek a programozásról

Mielőtt neki kezdenél a programozás megtanulásához vedd figyelembe ezeket a tippeket:

1. Első sorba ismerd fel a problémát majd gondolj ki egy megoldást és csak utána vágj bele a kódolásba.
2. Igyekezz minnél tisztább és rendezettebb kódot írni hogy olvasható legyen mindenki számára.
3. Spórolj minnél több memóriát azzal hogy olyan módszereket használsz amihez kevesebb kód kell.
4. Találd meg a neked megfelelő programozási nyelvet és fejlesztői környezetet.
5. Készíts minnél több olyan kódot amit újra felhasználható lesz.
